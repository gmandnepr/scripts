## Configure macro

Using dconf-editor:
* org/mate/macro/window-keybindings:
  * begin-mode->disabled
* org/mate/macro/global-keybindings:
  * switch_to_workspace_up->disabled
  * switch_to_workspace_down->disabled
  * switch_to_workspace_left->disabled
  * switch_to_workspace_right->disabled
